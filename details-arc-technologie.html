ARC Technologies

ARC Technologies est une entreprise française spécialisée dans les solutions électromécaniques avancées, avec un accent particulier sur les produits basse et moyenne tension. Elle est implantée à Langrolay-sur-Rance, en Bretagne, France.

Rôle : Stagiaire Développeuse en Business Intelligence

Analyser et visualiser les données des dépenses des clients des différentes agences

  • Collecter des données (SQL Server)

    • Logiciel : Microsoft SQL Server (T-SQL pour interroger et extraire les données).


  • Transformer des données (ETL - Extraction, Transformation, Chargement)

    • Logiciel : SQL Server Integration Services (SSIS).


  • Stocker des données

    • Logiciel : SQL Server (Tables, procédures stockées, vues).


  • Créer des rapports et des tableaux de bord pour BI

    • Logiciel : Power BI (avec Power Query et DAX pour les visualisations).


  • Diffuser des informations

    • Logiciel : Power BI (publication de rapports sur le service Power BI ou export en format PDF/Excel).
